# CadastroClientes

Aplicação ASP.NET Core MVC com SQLite e Entity Framework Core.

## ✨ Funcionalidades

- Cadastro de clientes
- Edição e exclusão
- Listagem com validação
- Banco de dados local com SQLite
- Scaffold automático de CRUD

## 🛠 Tecnologias

- ASP.NET Core MVC (.NET 6+)
- Entity Framework Core
- SQLite
- Bootstrap (opcional)

## 🚀 Como rodar

- Clone o projeto
- Restaure os pacotes
- Rode as migrações com `Update-Database`
- Execute com `Ctrl + F5`
